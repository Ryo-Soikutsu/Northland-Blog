# Secure The Network v2

V2 of Secure The Network is largely inspired by various HackTheBox sherlock challenges that I have attempted and completed as part of my WorldSkills training (mainly their OpTinselTrace24-x series of challenges). The full challenge and challenge questions will (hopefully) be quite large, covering a wide range of blue-team topics such as malware analysis, Windows Events analysis and more.

This is a current list of topics that I plan to include for STN-v2:

* Incident response and SOC
* Malware analysis
* Windows Event Logs analysis
* Forensics of common applications
* Threat intelligence

Each topic will be split into its separate sections, and given individual subpages
